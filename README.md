# easy webpack vue

webpack构建的一个vue.js学习、开发环境。

## 使用方法：

```
# 1. clone this repo
git clone https://github.com/xm2by/easy-webpack-vue.git

# 2. enter the easy-webpack-vue file
cd easy-webpack-vue

# 3. install dependencies
npm install

# 4. dev for development
npm run dev

# 5. build for production
npm run build
```
